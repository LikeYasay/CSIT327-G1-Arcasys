What is ARCASYS?

ARCASYS is a Marketing Archive Manager it is a web-based system designed to
centralize event creation, management, and tracking at Cebu Institute of Technology –
University. It ensures fast and secure access to event information,
automated categorization of past and recent events, and efficient retrieval of event links.

Centralized Database

All event information stored in one secure, accessible location with comprehensive search and filtering capabilities.

Fast & Secure Retrieval

Lightning-fast search with enterprise-grade security ensuring data integrity and authorized access only.

Automated Categorization

Smart categorization system automatically organizes events by type, date, department, and status for easy management.

The Problem We Solve

Understanding the current challenges and opportunities that drive the need for a modern event management solution.

Current Challenges

  • ⚠️ Lost or misplaced event links.
  • ⚠️ No centralized platform.
  • ⚠️ Hard to track past vs. upcoming events.
  • ⚠️ Inefficient manual announcements.

Opportunities Ahead

  • ✅ Real-time access to events.
  • ✅ Reliable, secure archive.
  • ✅ Streamlined coordination.
  • ✅ Improved transparency.

How It Works

1

Admins Add Events

Admins log in, create event records, and attach multiple links (FB, IG, TikTok, YouTube, Website).

2

System Organizes & Archives

Events are automatically categorized as Past or Upcoming, stored in a secure database with search & filters.

3

Viewers Retrieve & Export

Authorized users log in to search, view details, and export records with one click.
